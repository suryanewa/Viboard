# Viboard

Infinite canvas moodboard built with React, Vite, and Supabase.

## Scripts

- `npm run dev` — local development
- `npm run build` — production build
- `npm run lint` — ESLint

## Environment

Configure Supabase in `.env` (see `src/lib/supabase.ts`). Optional portfolio embed: set `VITE_PORTFOLIO_ATLAS=1` to load the Atlas demo board via `PortfolioApp`.
